# FitPick Voice and Copy

FitPick helps people know what to wear and get more from the clothes they already own. The voice is calm, refined, direct, and personal.

## Voice

- Simple: say the useful thing first.
- Confident: avoid hype and over-explaining.
- Warm: speak to the person, not the workflow.
- Private: be specific about wardrobe privacy without overstating security.
- Fashion-led: make dressing feel considered, not technical.

## Preferred Language

- Know what to wear.
- Wear more of what you own.
- Add a few pieces.
- Works with the weather.
- Fits the occasion.
- See why this works.
- Save this look.
- Try another look.
- Only you can see the wardrobe you add.

## Avoid

- revolutionize your wardrobe
- unlock your style potential
- AI-powered fashion ecosystem
- seamless experience
- game-changing
- smarter wardrobe
- elevate your fashion journey
- transform your wardrobe forever

Use "AI", "smart", "personalized", and "premium" only when the meaning would be unclear without them.

## Buttons

Buttons should be short and literal:

- Continue
- Send code
- Verify email
- Add item
- Save changes
- View outfit
- Try another
- Upload photo
- Start wardrobe

Avoid vague calls to action such as "Explore", "Discover", or "Experience now".

## Authentication

Use clear account language:

- Sign in
- Sign up
- Continue with email
- We will send a secure code.
- Only you can see the wardrobe you add.

New users should be prompted to sign up. Existing users should be prompted to sign in. Email OTP remains the primary authentication method.

## Errors

Make errors human and actionable:

- That code has expired. Request a new one and try again.
- That code does not look right. Check it and try again.
- We could not send a code right now. Please try again shortly.
- We could not read this photo clearly. Try a brighter image with the full item visible.

## Empty States

Empty states should tell the user what to do next:

- Your wardrobe starts here.
- Add a few pieces you wear often.
- Save outfits you want to wear again.

## Privacy

Use specific, supportable claims:

- Only you can see the wardrobe you add.
- You control what you upload.
- Remove any item at any time.
- Your photos are used to organize and style your wardrobe.

Do not claim zero-access storage, military-grade encryption, anonymity, or that images are never stored unless the implementation supports it.

## Capitalization

Use sentence case for headings, buttons, and labels unless a proper noun is required. Keep product terms consistent: FitPick, FitPick Plus, FitPick Stylist.
